### How to run this project 

```shell
# SSH clone this project 
> $ git clone git@github.com:ChengGuang-Li/my-portfolio.git

#enter the folder
> $ cd my-portfolio 

#install dependencies
> $ npm i 

#run 
> $ npm run dev 
```

### Sending email service

Enter the [EmailJS](https://www.emailjs.com/)  website. 

1. Create New Template --> Settings --> **Get email template id** 

2. Account --> **Get the public key**

3. Email Services --> **Get the service Id** 

Configure these three values in the **.env** file under the project.
```markdown
VITE_APP_EMAILJS_SERVICE_ID=    #service Id
VITE_APP_EMAILJS_TEMPLATE_ID=   #template id
VITE_APP_EMAILJS_PUBLIC_KEY=    #public key
```
